loc - Page 1

Global path to the page on which the step should work. Used to link a tour to a specific page.

Steps in which do not specify their own parameter "loc" will inherit the global parameter "loc"

If all steps of the tour should be shown only on one page, then it is enough to specify only the global parameter "loc".
